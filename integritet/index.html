Integritetspolicy

Vi samlar bara in uppgifter som behövs för att kunna svara på frågor, hantera bokningar, genomföra köp och följa upp serviceärenden.

Uppgifter vi kan behandla

Namn, telefonnummer, e-post, adress, fordonsmodell, beskrivning av fel, orderinformation och kommunikation med oss.

Varför

För att kunna boka tid, ge offert, genomföra betalning, leverera produkt, hantera reklamation och ge support.

Delning

Uppgifter kan delas med betalleverantör, fraktaktör eller teknisk driftleverantör när det krävs för tjänsten.

Dina rättigheter

Du kan kontakta oss för registerutdrag, rättelse eller radering när laglig grund inte längre finns.

Kontakt för integritetsfrågor: info@nordicemobility.se.
