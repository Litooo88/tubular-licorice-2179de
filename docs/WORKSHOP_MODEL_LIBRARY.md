# Workshop Model Library v0.1

Detta är en levande modellbank. Fält med okänd information ska fyllas på efter
verifierad verkstadserfarenhet.

## Ninebot G30

- Vanliga fel: punktering, bromsjustering, glapp i laddport, displayfel.
- Symptom: tappar kraft, skrapljud, broms tar dåligt, laddar inte.
- Felkoder: fyll på efter verifiering.
- Batteri/spänning: kontrollera originalspec innan batteriarbete.
- Däck/broms: vanligt med slang/däck och bromsjustering.
- Svagheter: slitage i bakhjul och broms.
- Felsökningsordning: visuell kontroll, däck, broms, laddare/laddport,
  felkod/app, provkörning.
- Reservdelar: däck/slang, bromsbelägg, laddport, display.
- Kundkommunikation: ge intervall tills diagnos är klar.
- Risknivå: normal, hög vid batteri/laddning.

- Reservdelar: slang/däck, bromsdelar, kablage, display.
- Kundkommunikation: förklara att räckvidd kan bero på batterihälsa.
- Risknivå: normal, hög vid batteri/fukt.

## E-Wheels E2S V2 Pro LR

- Vanliga fel: felkod E16, display/gas, controller, punktering.
- Symptom: E16 i display, startar inte, rycker, tappar drivning.
- Kända felkoder: E16 kräver diagnos innan slutpris.
- Batteri/spänning: kontrollera version och batterispec innan elarbete.
- Däck/broms: kontrollera däckdimension och bromstyp.
- Svagheter: elfel kan ha flera orsaker.
- Felsökningsordning: felkod, kablage/kontakter, display/gas, controller,
  motor, batteri.
- Reservdelar: display/gasreglage, controller, kablage, däck/slang.
- Kundkommunikation: startpris 395 kr, vanligt intervall 595-1995 kr, slutpris
  bekräftas innan arbete.
- Risknivå: medium, hög vid batteri/fukt.
